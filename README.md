# Резюме Антона Митенкова

Статичный сайт-резюме с возможностью переключения темы (светлая/темная) в стиле NothingOS.

## Установка и запуск

1. Склонируйте репозиторий
2. Откройте файл `index.html` в браузере

## Используемые технологии

- HTML5
- CSS3 (переменные, Grid, Flexbox)
- JavaScript (для переключения темы)

## Особенности

- Адаптивный дизайн
- Переключение между светлой и темной темой
- Сохранение выбранной темы в localStorage
- Минималистичный дизайн в стиле NothingOS

## Структура проекта
- index.html
- style.css
- script.js
